# \<kendo:dataSource-transport-destroy\>
A JSP tag representing Kendo Destroy.

#### Example
    <kendo:dataSource-transport>
        <kendo:dataSource-transport-destroy></kendo:dataSource-transport-destroy>
    </kendo:dataSource-transport>
