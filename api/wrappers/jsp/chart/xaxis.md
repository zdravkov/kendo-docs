# \<kendo:chart-xAxis\>

The X-axis configuration options of the scatter chart X-axis. Supports all valueAxis options.

#### Example
    <kendo:chart>
        <kendo:chart-xAxis></kendo:chart-xAxis>
    </kendo:chart>
