# \Kendo\Dataviz\UI\StockChartNavigatorCategoryAxisItemMajorTicks

A PHP class representing the majorTicks setting of StockChartNavigatorCategoryAxisItem.

#### Returns

#### Parameters

##### $value `string`



#### Example 
    <?php

    ?>
