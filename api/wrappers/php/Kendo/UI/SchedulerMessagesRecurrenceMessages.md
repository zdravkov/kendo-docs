# \Kendo\UI\SchedulerMessagesRecurrenceMessages

A PHP class representing the recurrenceMessages setting of SchedulerMessages.

`\Kendo\UI\SchedulerMessagesRecurrenceMessages`

#### Parameters

##### $value `string`



#### Example
