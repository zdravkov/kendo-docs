# \Kendo\UI\WindowPosition

A PHP class representing the position setting of Window.


## Methods

### left
Specifies the initial left position of the window.

#### Returns
`\Kendo\UI\WindowPosition`

#### Parameters

##### $value `float`



#### Example 
    <?php
    $position = new \Kendo\UI\WindowPosition();
    $position->left(1);
    ?>

#### Parameters
